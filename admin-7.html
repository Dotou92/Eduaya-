Configuration requise côté serveur
Pour que la suspension et la suppression de comptes fonctionnent, vérifie ces points dans Supabase / Vercel :
Ajoute une colonne suspendu (type boolean, défaut false) à la table profils.
Définis la variable d'environnement ADMIN_PWD sur Vercel avec le même mot de passe que celui utilisé ici.
SUPABASE_SERVICE_KEY doit déjà être configurée (utilisée aussi par les paiements) : elle permet de supprimer réellement le compte de connexion d'un élève.
La clé secrète FedaPay n'est plus exposée dans cette page : elle passe désormais par /api/admin-sponsoring, qui utilise FEDAPAY_SECRET_KEY côté serveur.
